# 使用
如何用好这个项目？

## 跟着练
1. 你可以直接下载到你本地
2. 阅读完前两章安装好 Python ，我们用的是 Anaconda，就是把它装好
3. 然后打开 Jupyter Notebook 跟着开始操作。
